## What is Mikoto?

Mikoto organizes communities into **spaces** — each space contains **channels** that can be used for text chat, voice/video calls, or collaborative document editing. Members join spaces via invite links or public handles, and administrators control access through a flexible role and permission system.

## Key features

- **Thread-based messaging** — Organize conversations with nested threads across text channels
- **Voice, video & screen sharing** — Real-time A/V powered by LiveKit
- **Collaborative wikis** — Document channels with live co-editing via Y.js
- **Bot API** — First-class SDK support in JavaScript (`@mikoto-io/mikoto.js`) and Python (`mikoto.py`)
- **Role-based permissions** — Granular permission flags assignable per role
- **Direct messages** — Threaded DMs and group DMs between users
- **Self-hostable** — Full Docker Compose setup; bring your own infrastructure

## Platform availability

Mikoto is available as a hosted service at [platform.mikoto.io](https://platform.mikoto.io) and as a self-hosted installation you deploy on your own servers.
